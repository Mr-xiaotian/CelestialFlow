# dashboard.css

> 📅 最后更新日期: 2026/05/23

负责仪表盘页面的核心三栏布局架构。

## 布局架构 (`.three-column-container`)

仪表盘采用宽屏三栏设计，通过 `flex` 布局实现页面的结构化展示：

- **左侧面板 (`.left-panel`)**: 占比 25%。通常放置“结构图”和“图分析信息”卡片。
- **中间面板 (`.middle-panel`)**: 占比 40%。核心区域，用于展示“节点运行状态”卡片网格。
- **右侧面板 (`.right-panel`)**: 占比 25%。放置“节点指标走向（折线图）”和“总体状态摘要”卡片。

## 页面显示逻辑

- **选项卡切换**: 通过 `.tab-content` 和 `.tab-content.active` 配合 JS 实现不同功能页（Dashboard / Errors / Injection）的瞬间切换。

## 响应式适配

- **单列回退**: 在宽度小于 `2048px` 时，布局自动从横向三栏切换为纵向单列流式布局，所有面板宽度自动撑满 100%，确保在小屏设备上也能正常阅读指标。

# injection.css

> 最后更新日期: 2026/05/23

负责“任务注入”页签的整体布局与交互组件样式，包括提示区域、节点选择列表、JSON 编辑器以及文件上传区域。

## 提示区域 (`.tip-section`)

- **设计**: 位于页面顶部，采用浅蓝色背景（`--cornflower-50`）和深蓝色左边框（`--cornflower-500`），起到强调提示的作用。
- **图标与文字**: 内部使用 `flex` 布局排列提示图标和细则文字。

## 节点选择区域

- **搜索框 (`.search-container`)**: 采用相对定位，内置搜索图标。输入框在获得焦点时会通过过渡效果增强视觉反馈。
- **节点列表 (`.node-list`)**:
  - 列表项支持悬停高亮。
  - **状态限制**: 对于已停止的节点（`.disabled-node`），通过 `opacity: 0.5` 和 `pointer-events: none` 禁用点击交互，防止无效注入。
- **已选节点 (`.selected-item`)**: 被选中的节点会以蓝色标签形式显示在下方列表，并配有移除按钮。

## 输入方式切换 (`.input-toggle`)

- **设计**: 采用胶囊型（Capsule）布局，背景为浅灰色。
- **交互**: 内部按钮通过 `.active` 类切换样式，模拟物理开关的视觉感（增加阴影和背景色切换）。

## 数据输入区域

- **JSON 编辑器 (`.json-textarea`)**:
  - 使用等宽字体（`monospace`），确保 JSON 缩进对齐清晰。
  - 支持垂直方向拉伸（`resize: vertical`）。
- **文件上传区 (`#file-input-section`)**:
  - 采用大面积虚线框设计，支持点击触发文件选择。
  - **成功反馈**: 文件读取成功后，下方会出现绿色的 `.file-info` 状态条显示文件名。

## 提交控制 (`.submit-section`)

- **提交按钮 (`.btn-submit`)**:
  - 页面最醒目的操作点，具有明显的投影和悬停位移效果。
  - **加载态**: 提交中会显示 `.spinner` 旋转动画，并禁用点击。
- **状态消息 (`.status-message`)**: 在按钮左侧实时反馈注入结果（成功显示绿色，失败显示红色）。

# 格式化工具测试 (test_format.py)

> 📅 最后更新日期: 2026/06/22

## 作用
验证 `celestialflow.utils.util_format` 中的通用格式化工具，确保在终端输出、日志记录和报告生成时，数据能以易读且美观的方式展示。

## 核心测试对象
- `format_repr`: 智能字符串缩略与转义工具。
- `format_table`: 字符终端表格生成工具。

## 关键测试流程
1. **字符串缩略 (format_repr)**:
   - 验证短字符串保持原样。
   - 验证长字符串被截断为 `前段...后段` 格式，且总长度符合预期。
   - 验证特殊字符（换行符、反斜杠）被正确转义，防止破坏日志格式。
2. **表格格式化 (format_table)**:
   - **基础功能**: 验证二维列表被转换为带边框的字符表格，并自动生成行列索引。
   - **自定义配置**: 验证自定义行名、列名以及不同对齐方式（left/right）的渲染效果。
   - **容错性**: 验证不规则嵌套列表（列数不一致）能通过 `fill_value` 填充对齐，空数据返回友好提示。

## 测试重点
- **边界对齐**: 确保表格边框在不同长度内容下依然对齐。
- **转义安全**: 确保 `\n` 等字符在 repr 模式下不会产生实际换行。
- **内存安全**: 测试用例使用小规模数据，确保格式化逻辑不会因为递归或复杂循环导致性能问题。

## 运行方式

```bash
# 全部执行
pytest tests/utils/test_format.py -v

# 仅运行字符串缩略测试
pytest tests/utils/test_format.py -k "repr" -v

# 仅运行表格格式化测试
pytest tests/utils/test_format.py -k "table" -v
```

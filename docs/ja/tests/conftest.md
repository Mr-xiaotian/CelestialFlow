# グローバルテスト設定 (conftest.py)

> 📅 最終更新日: 2026/05/23

## 目的
`tests/` ディレクトリ全体のルートレベル設定ファイルとして機能し、テスト環境に必要なグローバル変数と環境変数を初期化します。

## コア機能
- **環境変数の読み込み**: `dotenv.load_dotenv()` を自動的に呼び出し、プロジェクトルートディレクトリの `.env` ファイル内の設定（API キー、データベース接続文字列など）がテスト起動時に利用可能であることを保証します。

## 注意事項
- このファイルは Pytest によって自動的に認識されます。
- グローバルレベルの Fixture を追加する必要がある場合は、このファイル内で定義してください。

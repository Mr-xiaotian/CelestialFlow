# UtilsFormat

> 📅 最終更新日: 2026/04/22

`utils/util_format.py` は汎用フォーマットツールを提供します。

## 主要関数

- `format_repr`: オブジェクトの文字列表現を安全に切り詰めます。
- `format_table`: 二次元データをテキストテーブルとしてレンダリングします。
- `format_duration`: 秒数を可読な時間表現に変換します。
- `format_timestamp`: タイムスタンプを日時文字列に変換します。
- `format_avg_time`: 平均処理速度をフォーマットします。

# i18n.ts

> 📅 最終更新日: 2026/06/22

国際化モジュール。多言語（中国語、英語、日本語）翻訳データの維持、現在の言語状態管理、および DOM テキストの自動更新を担当します。

## 型定義

### `Lang`

サポートされる言語識別子：`"zh-CN"` | `"en"` | `"ja"`。

## グローバル変数

| 変数 | 型 | 説明 |
|------|------|------|
| `currentLang` | `Lang` | 現在選択中の言語識別子。デフォルト `"zh-CN"` |
| `translations` | `Record<Lang, Record<string, string>>` | すべての言語の翻訳辞書を含む |

## コア関数

### `setLang(lang)`

グローバル言語設定を切り替えます。

- **パラメータ**: `lang: Lang` - 対象言語識別子。
- **動作**: `currentLang` を更新し、`<html>` タグの `lang` 属性を同期変更します。

---

### `t(key, ...args)`

翻訳キーに基づいて現在の言語のテキストを取得し、シンプルなプレースホルダー置換をサポートします。

- **パラメータ**:
  - `key: string` - 翻訳キー名。
  - `...args: string[]` - `{0}`、`{1}`... を置換するためのパラメータリスト。
- **戻り値**: 翻訳後の文字列。対応するキーが見つからない場合はキー名自体を返します。
- **フォールバック機構**: 現在の言語にそのキーがない場合、`zh-CN` へのフォールバックを試みます。

---

### `applyI18nDOM()`

特定の `data-i18n` 属性を持つ要素を全ページでスキャンし、翻訳を適用します。

**サポートされる属性：**
- `data-i18n`: `textContent` を置換。
- `data-i18n-placeholder`: `placeholder` を置換。
- `data-i18n-title`: `title` 属性を置換。
- `data-i18n-aria-label`: `aria-label` 属性を置換。

また、この関数は `document.title` も同期更新します。

## 翻訳キーグループ

- `app.*`: アプリケーショングローバルテキスト（タイトル等）
- `settings.*`: 設定パネル関連
- `tab.*`: タブ名
- `card.*`: カードタイトル
- `summary.*`: 集計データラベル
- `errors.*`: エラーログ関連
- `injection.*`: タスク注入関連
- `status.*`: ノード状態関連
- `chart.*`: グラフ関連
- `interval.*`: 時間間隔オプション

## 使用例

### `t()` / `setLang()` / `applyI18nDOM()` の使用例

以下の例は国際化モジュールの 3 つのコア関数のブラウザコンソールでの使用法を示します：

```typescript
// 1. t()：現在の言語の翻訳テキストを取得
console.log(t("app.title"));       // "任务图监控系统" (現在 zh-CN)
console.log(t("settings.saveSuccess")); // "设置已保存"

// 2. t() はプレースホルダー置換をサポート
// 翻訳キー "injection.successBatch": "已成功提交 {0} 个节点草稿"
console.log(t("injection.successBatch", "3"));
// "已成功提交 3 个节点草稿"

// 3. t() のフォールバック機構
// 現在の言語（例：ja）にそのキーがない場合、zh-CN にフォールバック
// zh-CN にもない場合はキー名自体を返す
console.log(t("nonexistent.key"));  // "nonexistent.key"

// 4. setLang()：グローバル言語を切り替え
setLang("en");
console.log(currentLang);           // "en"
console.log(document.documentElement.lang); // "en"
console.log(t("app.title"));       // "Task Graph Monitor"

setLang("ja");
console.log(t("app.title"));       // "タスクグラフモニター"

// 5. applyI18nDOM()：全ページをスキャンして翻訳を更新
// HTML に以下があると仮定：
//   <h1 data-i18n="app.title"></h1>
//   <input data-i18n-placeholder="errors.searchPlaceholder">
//   <div data-i18n-title="settings.tooltip"></div>
//
// applyI18nDOM() 呼び出し後：
//   <h1>任务图监控系统</h1>
//   <input placeholder="搜索错误或任务">
//   <div title="设置"></div>

// 6. 完全な言語切り替えフロー
function switchLanguage(lang: Lang) {
    setLang(lang);                    // currentLang と <html lang> を更新
    applyI18nDOM();                   // すべての data-i18n 要素を更新
    document.title = t("app.title"); // ページタイトルを更新
}

// 英語に切り替え
switchLanguage("en");

// 中国語に切り替え
switchLanguage("zh-CN");

// 7. 翻訳キーの存在確認
function hasTranslation(key: string, lang?: Lang): boolean {
    const targetLang = lang || currentLang;
    return !!translations[targetLang]?.[key];
}

console.log(hasTranslation("app.title"));                 // true
console.log(hasTranslation("app.title", "en"));           // true
console.log(hasTranslation("app.title", "ja"));           // true
console.log(hasTranslation("nonexistent.key", "zh-CN"));  // false
```

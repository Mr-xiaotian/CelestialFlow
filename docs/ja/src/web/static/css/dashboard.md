# dashboard.css

> 最終更新日: 2026/05/23

ダッシュボードページの中核となる 3 カラムレイアウトアーキテクチャを担当します。

## レイアウトアーキテクチャ (`.three-column-container`)

ダッシュボードはワイドスクリーン 3 カラム設計を採用し、`flex` レイアウトでページの構造化表示を実現します：

- **左パネル (`.left-panel`)**: 比率 25%。通常「構造図」と「グラフ分析情報」カードを配置。
- **中央パネル (`.middle-panel`)**: 比率 40%。コアエリアで、「ノード実行状態」カードグリッドの表示に使用。
- **右パネル (`.right-panel`)**: 比率 25%。「ノード指標推移（折れ線グラフ）」と「全体状態サマリー」カードを配置。

## ページ表示ロジック

- **タブ切り替え**: `.tab-content` と `.tab-content.active` を JS と連携し、異なる機能ページ（Dashboard / Errors / Injection）の瞬間切り替えを実現します。

## レスポンシブ対応

- **単一カラムへのフォールバック**: 幅が `2048px` 未満の場合、レイアウトは自動的に横方向 3 カラムから縦方向の単一カラムフローレイアウトに切り替わり、すべてのパネル幅が自動的に 100% に拡張され、小画面デバイスでも指標を正常に閲覧できます。

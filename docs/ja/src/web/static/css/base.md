# base.css

> 最終更新日: 2026/05/28

システムのグローバル基本スタイル、ダークモード切替、共通コンポーネント（カード、タブ、バッジ）、レスポンシブ基本レイアウト、カードレイアウトエディターモーダルスタイルを管理します。

## グローバル基本

- **リセット**: ボックスモデルの統一（`border-box`）とデフォルトフォントシーケンス。
- **背景と色**: ライト（`--carbon-50`）およびダーク（`--carbon-900`）モードの body 背景色を定義。
- **コンテナ**: `.container` はコンテンツの最大幅を `1200px` に制限し水平中央揃え。

## コアコンポーネントスタイル

### ヘッダーとナビゲーション (`header`)
- **コントロールパネル (`.control-panel`)**: リフレッシュ間隔セレクター、設定歯車、テーマ切替ボタンを含む。
- **設定パネル (`.settings-panel`)**: 歯車アイコンの下に絶対位置でフローティング表示。言語切替、履歴制限、ページサイズなどの設定項目をサポート。

### タブシステム (`.tabs`)
- 水平配置のタブナビゲーションを実装。`.active` クラスで現在選択中のモジュール（ダッシュボード、エラーログ、タスク注入）をハイライト。

### 共通カード (`.card`)
- 統一された背景角丸（`1rem`）とシャドウ効果。
- **ホバーフィードバック**: ホバー時にわずかな上方向変位（`translateY(-2px)`）を発生。

### ステータスバッジ (`.badge`)
- 異なる実行状態のノード用の色付きブロックラベルを定義（例: `.badge-running` は緑）。

## 補助クラス

- **色クラス**: `.text-success`（緑）、`.text-error`（赤）、`.text-pending`（灰）、`.text-duplicate`（橙）などのクイック着色クラスを提供。
- **デルタクラス**: `.text-delta-*` シリーズはダッシュボードで薄い指標変化値を表示するために使用。
- **非表示**: `.hidden` クラスは JS で要素の表示/非表示を素早く制御するために使用。

## モーダルオーバーレイ (`.overlay`)

- **`.overlay`**: 固定位置の全画面半透明黒オーバーレイ（`rgba(0,0,0,0.4)`）、`z-index: 200`、中央 flex レイアウト。カードレイアウトエディターなどのモーダルウィンドウをホストするために使用。
- **`.overlay.hidden`**: オーバーレイを `display: none` に設定。JS と組み合わせてモーダルの表示/非表示を制御。

## カードレイアウトエディター (`.layout-editor` シリーズ)

カードレイアウトエディターはオーバーレイ上に浮かぶモーダルウィンドウとして表示され、3カラムダッシュボードカードのドラッグ＆ドロップ並べ替えをサポートします。主要なサブセレクター：

| セレクター | 説明 |
|-----------|------|
| `.layout-editor` | エディターのメインコンテナ：角丸白背景カード、`max-width: 700px`、縦方向 flex レイアウト |
| `.dark-theme .layout-editor` | ダークモード時に `--carbon-800` 背景を使用 |
| `.layout-editor-header` | タイトルバー：左右配置、タイトル左、閉じるボタン右 |
| `.layout-editor-title` | タイトル文字：`1.1rem`、`font-weight: 600` |
| `.layout-editor-columns` | 3カラムグリッドレイアウトエリア：`grid-template-columns: repeat(3, 1fr)`、縦スクロール可能 |
| `.layout-column` | 単一カラムコンテナ：縦方向 flex カラム |
| `.layout-column-header` | カラムタイトル：中央揃え、下線区切り、小フォント |
| `.layout-column-dropzone` | ドラッグ＆ドロップゾーン：破線境界、最小高さ `120px`、縦方向 flex でカード配置 |
| `.layout-column-dropzone.drag-over` | ドラッグホバー時にハイライト：青境界 + 薄青背景 |
| `.layout-card` | ドラッグ可能なカード項目：灰背景角丸、`cursor: grab`、`user-select: none` |
| `.layout-card:hover` | ホバー時に浮き上がりシャドウ効果 |
| `.layout-card.dragging` | ドラッグ中に半透明（`opacity: 0.5`） |
| `.layout-card-name` | カード名文字：`0.8rem`、`font-weight: 500` |
| `.layout-card-handle` | ドラッグハンドル：⠿ 文字、`color: --carbon-400` |
| `.layout-unused` | 未使用カードプールエリア：3カラムの下に配置 |
| `.layout-unused-header` | 未使用プールタイトル：`0.75rem`、灰色 |
| `.layout-unused .layout-column-dropzone` | 未使用プールドロップゾーン：水平配置（`flex-direction: row`）、最小高さ `40px` |
| `.layout-editor-footer` | フッターボタンバー：右揃え、上部分離線 |
| `.btn-layout-save` | 保存ボタン：青塗り、`80%` 幅 |
| `.btn-layout-reset` | リセットボタン：灰アウトライン、`20%` 幅 |
| `.btn-layout-editor` | 設定パネル内のエントリーボタン：青塗り、角丸 |

## レスポンシブルール

### `@media (max-width: 2048px)`

ビューポート幅 ≤ 2048px 時に以下の調整をトリガー：
- `h1` タイトル幅を `100%` に設定し、長いタイトルのオーバーフローを防止
- `#theme-toggle` テーマ切替ボタンを `position: static`、`order: 3` に変更し、狭い画面でのコントロールバーの再配置に対応

## ダークモード対応

ルートノード識別子として `.dark-theme` クラスを使用。このモードでは、システムが自動的に以下のプロパティを調整します：
- 背景色とメイン文字色。
- カードと設定パネルの背景色と境界色。
- フォームコントロール（select, button）の背景と境界。
- 一部の意味的文字色（pending 状態の文字色など）。

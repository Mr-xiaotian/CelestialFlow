# base.css

> 最終更新日: 2026/06/18

システムのグローバルベーススタイル、ダークモード切り替え、汎用コンポーネント（カード、タブ、バッジ）、レスポンシブベースレイアウト、およびカードレイアウトエディタのモーダルウィンドウスタイルを担当します。

## グローバルベース

- **リセット**: ボックスモデル（`border-box`）を統一し、デフォルトフォントシーケンスを設定します。
- **背景と色**: ライトモード（`--carbon-50`）とダークモード（`--carbon-900`）での body 背景色を定義します。
- **コンテナ**: `.container` はコンテンツの最大幅を `1200px` に制限し、水平中央揃えにします。

## コアコンポーネントスタイル

### ヘッダーとナビゲーション (`header`)
- **コントロールパネル (`.control-panel`)**: リフレッシュ間隔選択、設定ギア、テーマ切り替えボタンを含みます。
- **設定パネル (`.settings-panel`)**: 絶対配置でギアの下にフローティング表示され、言語切り替え、履歴制限、ページあたり件数などの設定項目をサポートします。

### タブシステム (`.tabs`)
- 横方向に配置されたタブナビゲーションを実装し、`.active` クラスで現在選択中のモジュール（ダッシュボード、エラーログ、タスク注入）をハイライトします。

### 汎用カード (`.card`)
- 統一された背景の角丸（`1rem`）とシャドウ効果を持ちます。
- **ホバーフィードバック**: ホバー時にわずかな上方変位（`translateY(-2px)`）が発生します。

### 状態バッジ (`.badge`)
- ノードの異なる実行状態におけるカラーブロックラベルを定義します（例：`.badge-running` は緑色）。

## ヘルパークラス

- **カラークラス**: `.text-success`（緑）、`.text-error`（赤）、`.text-pending`（灰）、`.text-duplicate`（オレンジ）などのクイックカラーリングクラスを提供します。
- **増分クラス**: `.text-delta-*` シリーズは、ダッシュボードで薄めの指標変化数値を表示します。
- **非表示**: `.hidden` クラスは JS で要素の表示/非表示を素早く制御します。

## モーダルオーバーレイ (`.overlay`)

- **`.overlay`**: 固定配置の全画面半透明黒オーバーレイ（`rgba(0,0,0,0.4)`）、`z-index: 200`、中央揃えのフレックスレイアウトで、カードレイアウトエディタなどのモーダルウィンドウを保持します。
- **`.overlay.hidden`**: オーバーレイを `display: none` に設定し、JS でポップアップの表示/非表示を制御します。

## カードレイアウトエディタ (`.layout-editor` シリーズ)

カードレイアウトエディタは、オーバーレイの上にモーダルウィンドウとしてフローティング表示され、3 カラムのダッシュボードカードのドラッグ再配置をサポートします。主要なサブセレクタ：

| セレクタ | 説明 |
|--------|------|
| `.layout-editor` | エディタのメインコンテナ：角丸白背景カード、`max-width: 700px`、縦方向 flex レイアウト |
| `.dark-theme .layout-editor` | ダークモードでは `--carbon-800` 背景を使用 |
| `.layout-editor-header` | タイトルバー：左右分布、タイトル左寄せ、閉じるボタン右寄せ |
| `.layout-editor-title` | タイトルテキスト：`1.1rem`、`font-weight: 600` |
| `.layout-editor-columns` | 3 カラムグリッドレイアウトエリア：`grid-template-columns: repeat(3, 1fr)`、縦スクロール可能 |
| `.layout-column` | 単一カラムコンテナ：縦方向 flex 列 |
| `.layout-column-header` | カラムタイトル：中央揃え、下線区切り、小フォント |
| `.layout-column-dropzone` | ドロップゾーン：破線ボーダー、最小高さ `120px`、縦方向 flex でカードを配置 |
| `.layout-column-dropzone.drag-over` | ドラッグホバー時にハイライト：青ボーダー + 薄青背景 |
| `.layout-card` | ドラッグ可能なカードアイテム：灰色背景角丸、`cursor: grab`、`user-select: none` |
| `.layout-card:hover` | ホバー時に浮き上がりシャドウ効果 |
| `.layout-card.dragging` | ドラッグ中は半透明（`opacity: 0.5`） |
| `.layout-card-name` | カード名テキスト：`0.8rem`、`font-weight: 500` |
| `.layout-card-handle` | ドラッグハンドル：⠿ 文字、`color: --carbon-400` |
| `.layout-unused` | 未使用カードプールエリア：3 カラムの下に配置 |
| `.layout-unused-header` | 未使用プールタイトル：`0.75rem`、灰色 |
| `.layout-unused .layout-column-dropzone` | 未使用プールドロップゾーン：横方向配置（`flex-direction: row`）、最小高さ `40px` |
| `.layout-editor-footer` | 下部ボタンバー：右揃え、上部区切り線 |
| `.btn-layout-save` | 保存ボタン：青塗りつぶし、幅 `80%` |
| `.btn-layout-reset` | リセットボタン：灰色枠線、幅 `20%` |
| `.btn-layout-editor` | 設定パネル内のエントリーボタン：青塗りつぶし、角丸 |

## レスポンシブルール

### `@media (max-width: 2048px)`

ビューポート幅 ≤ 2048px で以下の調整が発動します：
- `h1` タイトル幅を `100%` に設定し、長いタイトルのオーバーフローを防止
- `#theme-toggle` テーマ切り替えボタンを `position: static`、`order: 3` に変更し、狭い画面でのコントロールバー再配置に対応

## ダークモード対応

`.dark-theme` クラスをルートノード識別子として採用。このモードでは、システムが自動的に以下のプロパティを調整します：
- 背景色とメインテキスト色。
- カードと設定パネルの背景とボーダー色。
- フォームコントロール（select、button）の背景とボーダー。
- 一部の意味的テキスト色（pending 状態のテキスト色など）。

# dashboard_analysis.css

> 📅 最終更新日: 2026/05/23

ダッシュボード左下の「グラフ分析情報」カードのスタイル定義を担当します。

## レイアウト設計 (`.analysis-info`)

- **構造**: 縦方向 `flex` レイアウトでキー・バリューペアのリストを表示します。
- **フォント**: 小さいフォントサイズ（`0.75rem`）を使用し、より多くのメタデータ情報を収容します。

## データ行スタイル (`.analysis-row`)

- **左右揃え**: ラベル名（Label）は左寄せ、具体的な数値（Value）は右寄せ。
- **状態色 (`.analysis-value`)**:
  - `.ok`: 緑（`--jade-600`）、期待通りであることを示します（例：DAG である）。
  - `.warn`: 赤（`--crimson-600`）、潜在的なリスクを示します（例：循環が存在する）。

## 関連モジュール

- データレンダリングは `dashboard_analysis.ts` が担当し、このスクリプトはバックエンドから返された分析結果（DAG かどうか等）に基づいて動的に `ok` または `warn` クラスを割り当てます。

# errors.css

> 📅 最終更新日: 2026/05/23

「エラーログ」タブの検索ボックス、フィルター、データテーブル、ページネーションコントロールのスタイル定義を担当します。

## 検索とフィルター (`.filter-container`)

- **レイアウト**: `flex` レイアウトを使用し、左側にタイトル、右側に検索入力ボックスとノードドロップダウンフィルターを表示します。
- **検索ボックス (`.error-search-input`)**:
  - スムーズなトランジション効果（`transition: all 0.2s`）を持ちます。
  - **フォーカス状態**: フォーカス取得時にボーダー色が濃くなり、背景色がわずかに変化して視覚的フィードバックを提供します。

## データテーブル (`#errors-table`)

- **構造**: 標準 HTML テーブルですが、モバイル/狭い画面では自動的にカードフローレイアウトに変換されます。
- **セルスタイル**:
  - `.error-id`: 灰色調で ID を表示し、視覚的ノイズを低減します。
  - `.error-cell`: エラー repr の表示専用で、等幅フォント（`monospace`）と赤色調（`Crimson`）を使用します。超長いエラー情報でレイアウトが崩れるのを防ぐため、`max-width: 40ch` を設定し省略記号（`ellipsis`）で切り詰めます。

## ページネーションコントロール (`.pager-container`)

- **ページ番号リンク (`.pager-link`)**: ホバーハイライトをサポートします。
- **ページめくりボタン (`.pager-btn`)**:
  - 「前のページ」と「次のページ」のアイコンボタンを含みます。
  - **無効状態**: 最初のページまたは最後のページの場合、ボタンは灰色になりクリック不可（`cursor: not-allowed`）になります。

## レスポンシブデザイン

- **テーブルからカードへ**: 幅が `2048px` 未満の場合、テーブル `thead` は非表示になり、各行 `tr` は独立した角丸ボーダーのカードブロックに変わります。
- **疑似要素ラベル**: `::before { content: attr(data-label) }` によりテーブル列名をカード内部に注入し、モバイルユーザーでもデータの意味を識別できるようにします。
- **エラー展開**: カードモードでは、`.error-cell` の切り詰め制限が解除され（`white-space: normal`）、自動折り返しに変更されてより多くの詳細を表示します。

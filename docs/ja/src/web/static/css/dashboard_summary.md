# dashboard_summary.css

> 最終更新日: 2026/05/23

ダッシュボード右下の「全体状態サマリー」パネルのスタイル定義を担当し、鮮やかな色使いの統計カードでグローバル実行指標を表示します。

## レイアウト構造 (`.summary-grid`)

- `grid` レイアウトを使用し、固定で 2 列に設定。
- 各グリッドセルは 1 つの `.summary-item` を含みます。

## 統計カード (`.summary-item`)

各統計項目はその意味に応じて異なるテーマ色が割り当てられ、背景色（`.summary-item`）と数値テキスト色（`.summary-value`）を含みます：

| 統計項目 | クラス名 | メインカラー | 意味 |
|--------|------|--------|------|
| **総成功タスク** | `.success` | `Jade`（緑） | タスクが正常に完了 |
| **総待機タスク** | `.pending` | `Carbon`（灰） | キュー内で処理待ち |
| **総エラータスク** | `.error` | `Crimson`（赤） | 処理失敗、注意が必要 |
| **総重複タスク** | `.duplicate` | `Marigold`（橙） | 重複排除ロジックにヒット |
| **アクティブノード** | `.nodes` | `Cornflower`（青） | 現在実行中の Stage 数 |
| **総残り時間** | `.remain` | `Violet`（紫） | グローバル進捗推定 |

## スタイル特性

- **視覚階層**: 数値は `2rem` 太字フォント、ラベルは `0.75rem` 灰色小文字を使用。
- **ダークモード**: 背景色を自動的に暗いトーン（`900` シリーズ）に切り替え、テキスト色を明るいトーン（`300` シリーズ）に切り替えてコントラストを確保します。
- **インタラクション示唆**: 
  - 総エラー数が 0 より大きい場合、`.summary-value.error-clickable` にポインターカーソルが表示され、クリックでエラー詳細へジャンプできることをユーザーに示します。

# dashboard_structure.css

> 📅 最終更新日: 2026/05/23

タスク構造図（Mermaid.js レンダリングコンテンツ）のコンテナレイアウトとテーマ対応を担当します。

## コンテナレイアウト (`#mermaid-container`)

- **配置方法**: `flex` レイアウトでコンテンツの水平・垂直中央揃えを実現します。
- **スクロールサポート**: `overflow-x: auto` を有効にし、大規模で複雑なグラフ構造でもユーザーが横スクロールで完全なトポロジーを閲覧できるようにします。

## Mermaid テーマ対応

Mermaid がレンダリングする SVG の内部スタイルは通常の CSS で上書きするのが難しいため、本ファイルでは主にダークモード（`.dark-theme`）向けの重要なスタイル強制上書きを行っています：

- **矢印色 (`.arrowMarkerPath`)**: ダークモードで矢印の塗りつぶし色を `--carbon-200` に設定し、接続線の視認性を向上させます。
- **ノードテキスト (`span`)**: ノード内部のテキスト色を強制的に `--carbon-300` に変更し、暗い背景ノードでの可読性を確保します。

## 関連モジュール

- 具体的なノード背景色、ボーダー色、接続線スタイル（`classDef`）は、`dashboard_structure.ts` 内で現在のテーマに基づいて動的に生成される Mermaid コードで定義され、この CSS ファイルには含まれません。

# UtilsFormat

> 📅 Last Updated: 2026/04/22

`utils/util_format.py` provides general-purpose formatting tools.

## Main Functions

- `format_repr`: Safely truncates object strings.
- `format_table`: Renders two-dimensional data as a text table.
- `format_duration`: Converts seconds to human-readable duration.
- `format_timestamp`: Converts timestamps to date-time strings.
- `format_avg_time`: Formats average processing speed.

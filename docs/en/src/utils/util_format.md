# UtilsFormat

> 📅 Last updated: 2026/04/22

`utils/util_format.py` provides general-purpose formatting tools.

## Main Functions

- `format_repr`: Safely truncates an object's string representation.
- `format_table`: Renders two-dimensional data as a text table.
- `format_duration`: Converts seconds to a human-readable duration.
- `format_timestamp`: Converts a timestamp to a datetime string.
- `format_avg_time`: Formats average processing speed.

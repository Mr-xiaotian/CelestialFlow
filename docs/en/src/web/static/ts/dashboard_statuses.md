# dashboard_statuses.ts

> 📅 Last Updated: 2026/06/05

## Purpose

Explains how the dashboard loads node status snapshots, computes card deltas, colors elapsed duration, and renders sortable status cards.

## Key Points

- Defines the `NodeStatus` snapshot structure used across the dashboard.
- Highlights the colored elapsed-time renderer and progress-bar segmentation.
- Covers status polling and Sortable.js-based card ordering.

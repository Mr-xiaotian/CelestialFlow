# globals.d.ts

> 📅 Last Updated: 2026/06/05

## Purpose

Documents the global TypeScript declarations used by CDN-loaded libraries and cross-module helpers.

## Key Points

- Covers `Chart`, `Sortable`, `window.mermaid`, `Lang`, `currentLang`, `setLang`, `t`, and `applyI18nDOM`.
- Keeps non-module globals visible to the TypeScript compiler.
